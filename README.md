# Facebook Like Box Style Code

[![CSS](https://img.shields.io/badge/CSS-snippet-1572B6?logo=css3&logoColor=white)](#usage-historical)
[![Status](https://img.shields.io/badge/status-legacy%20archive-lightgrey)](#status)

Customize the layout of the Facebook Like Box with simple CSS styles and
remove the Facebook branding from the like box.

## Status

**Legacy archive (2013).** Facebook retired the Like Box widget in 2015 and
replaced it with the Page Plugin, so this snippet no longer works against
Facebook's current embeds. It is kept for reference only and is not maintained.

## Usage (historical)

1. Add the markup from `disply-fb-box.txt` where the box should appear, and
   replace `YOUR FACEBOOK PAGE URL HERE` with your page URL.
2. Add the rules from `style.css` to your theme's stylesheet. The
   `.TechLilaFB` wrapper crops the widget to a compact 295px card and hides
   the Facebook branding.
